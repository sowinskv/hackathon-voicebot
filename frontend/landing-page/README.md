# Super Kitties - Landing Page

Minimal landing page for the Voice Banking Platform with navigation to all three apps.

## Structure

```
landing-page/
├── index.html           # Main landing page with inline styles
├── public/
│   └── ui-assets/
│       └── -59.jpg     # Background image
└── README.md
```

## Features

- Glass morphism cards for each app
- Blurred background (40px)
- Smooth fade-in animations
- Ambient floating glow particles
- Responsive grid layout
- Minimal aesthetic matching the rest of the platform

## Access

- **Docker**: http://localhost:8000
- **Standalone**: Open `index.html` directly in browser

## Apps

- 🎙️ **Voice Banking Assistant** - http://localhost:5173
- 🤖 **Bot Builder** - http://localhost:5174
- 📊 **Agent Console** - http://localhost:5175
