QUICK START
===========

1. Install dependencies:
   npm install

2. Configure environment:
   cp .env.example .env
   # Edit .env with your credentials

3. Setup database:
   psql -U postgres -d voicebot -f database.sql

4. Run service:
   npm run dev

5. Test it:
   ./example-usage.sh
